# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-01

### Added
- Initial release
- Event tracking functionality
- User identification
- Automatic event batching
- localStorage persistence
- Retry logic for failed requests
- Auto-tracking for page views and clicks
- TypeScript support
- Comprehensive test suite
- Example demo page

### Features
- Event queue with configurable batch size
- Automatic flush on interval
- Manual flush support
- Session management
- User persistence
- Debug logging mode
- sendBeacon fallback for page unload

## [Unreleased]

### Planned
- Support for custom storage adapters
- Plugin system
- More auto-tracking options
- Performance monitoring
- Error tracking
- A/B testing integration
